# Hostel Management System

## 📌 Project Overview
The Hostel Management System is a Java and MySQL-based desktop application developed to simplify hostel administration.

## ✨ Features
- Student Registration
- Student Login
- Admin Login
- Room Allocation
- Fee Management
- Search Student
- Update Student Details

## 🛠️ Technologies Used
- Java
- Java Swing
- JDBC
- MySQL
- Eclipse IDE

## 🚀 How to Run
1. Import the project into Eclipse.
2. Configure MySQL and create the database.
3. Add the MySQL Connector (JDBC).
4. Run Main.java.
